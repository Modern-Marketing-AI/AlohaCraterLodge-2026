PROJECT OVERVIEW:
Build a static, high-conversion Itinerary Planner and an updated FAQ section for Aloha Crater Lodge. 

ENVIRONMENT & STYLING RULES:
1. Framework: Pure HTML5, Vanilla JavaScript, and Tailwind CSS (via CDN). No heavy front-end frameworks like React or Vue.
2. Layout Architecture: The itinerary planner must use a split-pane layout on desktop (40% left interaction, 60% right visualization) and a progressive vertical stack on mobile. 
3. Sizing Constraint: Use `100dvh` for the main wrapper and `flex-shrink-0` on critical bottom elements so mobile keyboards do not warp the timeline during email capture.
4. UI Visibility: All elements must remain permanently visible. Do not use hidden drawers, <details> tags, or collapsible accordions.
5. Color Palette: Primary background is Deep Emerald/Moss Green (safety/sanctuary), Secondary is Luxury Slate/Charcoal (premium night framing), and Action Accents are Volcanic Amber (strictly for buttons/CTAs).
6. Integrations: The final booking button must construct a URL string appending `?checkin=YYYY-MM-DD&checkout=YYYY-MM-DD` to the CloudBeds variable.

BRAND VOICE & CONTENT RULES:
1. The tone is "Lodge Guide": Warm, professional, and direct.
2. BANNED WORDS: Cozy, nest, healing, pamper, hidden gem, yummy, protocols, tactical, deployment, spa. 
3. BANNED TOPICS: Never mention or code any references to the on-property lava tube or caves.
4. EXACT NOMENCLATURE: Optical therapy equipment must be named "OlyLife Galaxy G-One". The recovery area is the "Bio-Regeneration Recovery Room".
5. ASSETS: Do not use or code any <img> placeholder tags. Visual cards will render as clean, colored blocks until the verified media files are delivered by the owners.

    function updateState(key, value) {
      state[key] = value;
      renderTimeline();
    }

    function renderTimeline() {
      const container = document.getElementById('timeline-container');
      container.innerHTML = ''; 

      // Inject Intercept Warning
      if (state.intercept === 'kona') {
        container.innerHTML += contentBlocks.intercept_warning;
      }

      // Inject Standard Blocks
      if (state.vibe || state.duration || state.checkin || state.intercept) {
        container.innerHTML += contentBlocks.morning_base;
        if (state.vibe === 'lava') container.innerHTML += contentBlocks.afternoon_lava;
        container.innerHTML += contentBlocks.evening_curfew;
      } else {
        // Fallback to reload manifesto if somehow empty but triggered
        container.innerHTML = document.getElementById('manifesto-block').outerHTML;
      }
    }

    function executeBooking() {
      if (!state.checkin || !state.checkout) {
        // Fallback Loop: Send to general booking if dates aren't selected
        window.location.href = CLOUDBEDS_URL;
        return;
      }
      // Construct CloudBeds exact parameter string
      const deepLink = `${CLOUDBEDS_URL}?checkin=${state.checkin}&checkout=${state.checkout}&adults=2`;
      window.location.href = deepLink;
    }
